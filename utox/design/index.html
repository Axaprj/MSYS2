uTox compact design (draft)

Purpose

Visual modelling of compact uTox interface. The sketches should be suitable as a convenient option of desktop interface and should allow to use the uTox on most portable devices.
The main solution to make the UI smaller is avoid a two-panel interface.
Contacts-Chat solution:
  • Connection state of a contact person may be displayed in more more elegant way, colored background bar for example.
  • Connection state of the client imho is useful to displays in a title bar.

Settings solution:
